# hubot-oyatsu

A hubot script that check oyatsu

See [`src/oyatsu.coffee`](src/oyatsu.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-oyatsu --save`

Then add **hubot-oyatsu** to your `external-scripts.json`:

```json
[
  "hubot-oyatsu"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
